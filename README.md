# simple-mapbox-react-flask-gdal

# Quick-Start Local Setup

1. clone down the repo:
```git clone https://github.com/dylandangphu/simple-mapbox-react-flask-gdal.git```

2. build:
```docker-compose build```

2. run:
```docker-compose up```

3. UI - navigate to url:
```http://localhost:3000/```

4. kill:
```docker-compose down```
